# tabletop-scheduler

Per il funzionamento della messaggistica Telegram, copiare config.ini.esempio in config.ini e inserire i parametri corretti.
BotToken è la stringa ottenuta da @botfather.
GroupID è l'identificativo telegram del gruppo in cui si vuole mandare il messaggio alla creazione di una partita. Si può ottenere con
Perché l'invio funzioni, il bot deve essere aggiunto al gruppo.
